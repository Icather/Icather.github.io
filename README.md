# 一氯氢化物的博客

> 高中生 / 独立开发者 / Android & Web

我的个人主页与博客，托管于 GitHub Pages。

## 🔗 访问地址

[https://icather.github.io](https://icather.github.io)

## 📁 目录结构

```
├── index.html          主页
├── notes.html          文章独立阅读页
├── start-server.bat    本地预览启动脚本
└── static/
    ├── css/            样式文件
    ├── js/             脚本文件
    ├── fonts/          字体资源
    ├── img/            图片资源
    └── md/             博客文章 (Markdown)
```

## 🖥️ 本地预览

双击运行 `start-server.bat`，访问 `http://localhost:8080` 即可预览。

需要 Python 或 Node.js 环境。
